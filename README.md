# Complaint Management Corner

A full-stack web application for managing complaints. Users can register, log in, submit complaints, track their status, and administrators can manage and resolve complaints through a centralized dashboard.

## 🚀 Features

- User Authentication
- Complaint Registration & Tracking
- Admin Dashboard
- CRUD Operations
- Responsive Design

## 🛠 Tech Stack

**Frontend:** React.js, HTML, CSS, JavaScript, Java  
**Backend:** Node.js, Express.js, SpringBoot  
**Database:** MySQL  
**Tools:** Git, GitHub, Postman

## 📂 Installation

```bash
git clone https://github.com/BhanjaMihirKumar/CMC-Complaint-Management-Corner.git

cd Complaint-Management-System

npm install

npm start
```
